hexo配置404界面

发表于 2019-11-26 分类于 hexo

本文主要讲解hexo下的404界面配置
如果将404.html直接放在根目录的source文件夹下，在编译的时候会改变文本内容

所以改为放在你配置的主题文件夹下的source文件夹内

在编译的时候会直接生成到public下,可以保持原本的内容
